# 🧾 HR Payroll Management System 
ระบบบริหารจัดการทรัพยากรบุคคลและเงินเดือน (HR & Payroll)  
ออกแบบมาเพื่อช่วยองค์กรจัดการข้อมูลพนักงาน การลา การเข้า-ออกงาน และการคำนวณเงินเดือนแบบครบวงจร  
## Features 
### 👩‍💼 ฝั่งผู้ดูแลระบบ (HR / Admin)
### 📊 Dashboard
- แสดงกราฟสรุปข้อมูลพนักงานรายวัน  
- จำนวนพนักงานเข้างาน  
- จำนวนพนักงานลา  
- จำนวนพนักงานมาสาย  
- สถิติการลาออกของพนักงานรายเดือน  
- กราฟแสดงจำนวนพนักงานแยกตามแผนก  
### 📄 ระบบอนุมัติการลา (Leave Approval Workflow)
- แสดงสถานะการลา - ส่งคำขอ → รอหัวหน้าอนุมัติ → รอ HR อนุมัติ → อนุมัติสำเร็จ
### 📥 การจัดการข้อมูลเวลาเข้างาน
- รองรับการนำเข้าข้อมูลจากไฟล์ CSV (เครื่องสแกนนิ้ว)
- แสดงข้อมูลแบบ ปฏิทินรายบุคคล
- แสดงวันมาสาย / ลา / ขาดงาน
- รองรับพนักงานแบบมีกะ (Shift Work)
### ⏱ ระบบ OT (Overtime)
- พนักงานสามารถยื่นคำขอ OT
- เชื่อมต่อกับระบบคำนวณเงินเดือน
### 💰 ระบบเงินเดือน (Payroll)
- คำนวณเงินเดือนอัตโนมัติ
- แสดงตัวอย่างสลิปเงินเดือนก่อนดาวน์โหลด
- Export สลิปเงินเดือนเป็น PDF
### ⚙️ ตั้งค่าระบบ
- ตั้งค่าวันหยุดตามเทศกาล
- กำหนดสิทธิ์วันลาของพนักงาน
- ตั้งค่าเบี้ยขยัน (กรณีไม่ขาด/ลา/สาย)
- จัดการกะการทำงาน
- ตั้งค่าข้อมูลบริษัท
### 👨‍💻 ฝั่งพนักงาน (Employee)
- Mobile Web Application
- เช็คเวลาเข้า-ออกงาน
- ขอลางาน
- ตรวจสอบสถานะการลา
### 👨‍💼 ฝั่งหัวหน้า (Manager)
- Mobile Web Application
- เช็คเวลาเข้า-ออกงาน
- อนุมัติการลาของลูกทีม
- ตรวจสอบข้อมูลพนักงานในทีม
### ✨ แนวคิดในการออกแบบ
- ออกแบบระบบแบบ Role-based (HR / Manager / Employee)
- เน้น UI ที่ใช้งานง่าย ลดความซับซ้อนของข้อมูล HR
- ใช้ Dashboard และ Data Visualization เพื่อช่วยในการตัดสินใจ
- รองรับการใช้งานทั้ง Mobile และ Desktop (Responsive Design)
- ออกแบบ Workflow ให้ชัดเจน เช่น การอนุมัติการลาแบบหลายขั้นตอน
### 🚀 จุดเด่นของโปรเจค
- พัฒนาระบบเองทั้งหมดตั้งแต่ต้นจนจบ (End-to-End)
- ครอบคลุมทั้ง Frontend, Backend และ UI Design
- รองรับการใช้งานจริงในองค์กร (Real-world Use Case)
### 🛠️ Tech Stack  
- Frontend: React, Ant Design  
- Backend: Node.js, Express  
- Database: Maria DB
